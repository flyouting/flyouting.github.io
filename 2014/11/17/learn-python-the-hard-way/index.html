常见问题

Q:这个课程需要多久学完?

你需要花费通读这本书的时间，并且每天都要集中精力完成习题。有些人用3个月时间，有的人要用6个月，也有的人只需要用一周的时间就足够了。 如果我不进行习题的研究的话，我可以用4小时或者更少的时间做完这本书的习题。

Q:我需要什么样的电脑?

你可以使用任何种类的计算机. 第一个练习里介绍了python在windows、Mac OSX、和Linux 上是如何运行的。
