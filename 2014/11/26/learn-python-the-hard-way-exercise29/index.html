Python

下面是你要写又一个python脚本，这节练习会向你介绍“if语句”。把这段代码输入脚本，让它正常运行，看看你有没有什么收获。

常见问题

Q: +=表示什么意思？

代码x += 1和x = x + 1 实现的是一样的功能，但是可以少输入一些字符。你可以称之为“增量”操作符。-= 也是相同的，后面你会看到更多的相关解释。
