常见问题

Q:为什么在后面你把 jelly_beans 这个变量叫做 beans?

这是函数如何工作的一部分。记住，在函数中，变量都是临时的.当你返回一个变量的时候，你可以把它赋值给另一个变量。我只是定义了一个叫做‘beans’的新变量去接收返回值。

Q:你所说的反向阅读代码是什么意思?

从最后一行开始阅读。对比你的代码和我的是不是一样。如果确认一样，向上移动一行阅读，直到你读到脚本的第一行为止。
